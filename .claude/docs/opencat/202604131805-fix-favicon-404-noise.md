## 变更动机

本地 Vite 开发（默认端口 1430）时，浏览器会隐式请求 `/favicon.ico`，仓库未提供该静态资源，导致无意义的 HTTP 404 与控制台噪音，干扰 Playwright 与人工调试。

## 变更范围

- 新增 `public/favicon.ico`（最小合法 ICO，由构建复制到 `dist/`）。
- 在 `index.html` 中增加 `<link rel="icon" href="/favicon.ico" />`。
- OpenSpec：`reference-app-shell-alignment` 主规格增加「常规 favicon 可访问」要求；变更已归档至 `2026-04-13-fix-favicon-404-noise`。
