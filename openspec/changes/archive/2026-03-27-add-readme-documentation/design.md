## Context

飞猫助手（FlyCat）是一个基于 Tauri 2.x + React 19 + TypeScript 的桌面应用，用于同步飞书知识库文档到本地。项目已完成核心功能开发，但缺少项目介绍文档。

## Goals / Non-Goals

**Goals:**
- 创建清晰的 README.md 文档，介绍项目功能和使用方法
- 包含技术栈、安装指南、开发环境配置
- 提升项目可发现性

**Non-Goals:**
- 不涉及代码修改
- 不创建详细的 API 文档
- 不创建用户手册

## Decisions

### 文档结构

采用标准开源项目 README 结构：
1. 项目名称和简介
2. 功能特性列表
3. 技术栈
4. 快速开始（安装和使用）
5. 开发指南
6. 项目结构
7. 许可证

### 命名

项目正式名称：飞猫助手（FlyCat）
- 中文名：飞猫助手
- 英文名：FlyCat
- 包名：feishu-docs-sync

## Risks / Trade-offs

无重大风险。文档内容需要随项目迭代更新。
