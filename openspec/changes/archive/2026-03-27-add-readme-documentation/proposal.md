## Why

飞猫助手（FlyCat）是一个用于同步飞书知识库文档到本地的桌面应用，目前项目缺少 README 文档，导致新用户和开发者难以快速了解项目的用途、功能特性和使用方法。需要创建一份完整的 README 文档来介绍项目。

## What Changes

- 新增 `README.md` 项目文档，包含以下内容：
  - 项目介绍和核心功能
  - 技术栈说明
  - 安装和使用指南
  - 开发环境配置
  - 项目结构说明

## Capabilities

### New Capabilities

- `readme-documentation`: 项目 README 文档，介绍飞猫助手的功能、安装和使用方法

### Modified Capabilities

无现有 capabilities 被修改，这是纯文档变更。

## Impact

- 新增文件：`README.md`
- 不影响现有代码和功能
- 提升项目可发现性和用户体验
